# jj-crawler
 晋江文章数据爬虫

# Dependency
- requests
- json
- random
- bs4

# 项目简介
晋江文学城，文章爬虫，本案例仅包含文章标题，文章详情以及文章类目。更多信息可自行通过HTML Tag进行增加。

**此爬虫仅供学习使用:)如有侵权，联系删除**

# 项目结构
- `jj_article_basic_info.py`：文章爬虫
